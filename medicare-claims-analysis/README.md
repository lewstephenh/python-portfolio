# Medicare Claims Analysis

## Description
Conducted an end-to-end quantitative analysis of Medicare beneficiary and claims data using Python, including data cleaning, validation, payment calculations, descriptive analyses, and multivariable regression to examine drivers of Medicare spending. Demonstrates Medicare claims and beneficiary data structure; health care payment components (Medicare, beneficiary responsibility, primary payer); quantitative health services research methods; data cleaning, merging, and validation; large-dataset analysis using Python (pandas, NumPy); statistical modeling (OLS regression with statsmodels); exploratory data analysis and visualization; creation of publication-ready tables and figures; technical documentation of methods and limitations; attention to data quality and analytic rigor.
